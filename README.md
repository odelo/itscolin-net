# itscolin.net

Simple static personal website built with HTML, CSS, and JavaScript.

## Structure

- `index.html`: One-page landing page with profile and project wheel.
- `styles.css`: Shared light theme styles.
- `script.js`: Horizontal wheel behavior for projects.
- `projects/*.html`: Individual project pages.

## Run locally

Open `index.html` directly in your browser, or serve with any static server.
